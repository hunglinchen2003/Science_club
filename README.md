# 探索科學，創新生活 — 教師科學社團

單頁網站，呈現社團的熱情與專業，鼓勵科學愛好者加入。

## 使用方式

1. 用瀏覽器開啟 `index.html` 即可預覽
2. 或使用本地伺服器（如 `npx serve .`）進行開發

## 自訂設定

- **GitHub 連結**：在 `index.html` 中搜尋 `href="https://github.com"`，改為您的 GitHub 專案網址
- **電子郵件**：預設為 `contact@science-exploration.org`，可依需求修改

## 技術說明

- 純 HTML + CSS，無需額外依賴
- 響應式設計，支援手機與桌面
- 使用 Noto Sans TC 字體（Google Fonts）
